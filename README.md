## Authors

## Install
```bash
git clone https://github.com/larsherbold/QFT-and-Conversion-from-Classical-to-Quantum-Circuits.git
cd QFT-and-Conversion-from-Classical-to-Quantum-Circuits
```

```bash
python3 -m venv venv
```

* On Linux/macOS:
```bash
source venv/bin/activate
```
* On Windows (PowerShell):

```powershell
.\venv\Scripts\Activate.ps1
```

```bash
pip install -r requirements.txt
```
